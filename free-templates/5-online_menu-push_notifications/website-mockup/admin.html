Recommended n8n workflow: Webhook → Telegram or → Email

If set, the menu is POSTed on every save (updates menu-data.js on the server).

How to configure the menu save webhook
  1. In n8n create a new workflow
  2. Add a Webhook node → Method: POST
  3. Add a Code node to transform the JSON into: window.MENU_DATA = {...};
  4. Connect a Write Binary File node with the absolute path of menu-data.js on the server
  5. Add a Respond to Webhook node
  6. Activate the workflow and paste the URL above
